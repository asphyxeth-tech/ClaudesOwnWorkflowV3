# Stream H — Finalist Deep-Dive Verdicts (2026-06-01)

Two finalists from the "α-ramp → β" lane got the deep adversarial dive. **Both came back
CONDITIONAL BET — the first survivors of the entire process.** Shared honest caveat: in
both, **AI is a sharp *edge*, not an irreplaceable hero**, and both are **crowded** → the
moat is *narrow niche + execution + a public, verifiable track record*, not novelty.

## H2 — "Toll Booth": builder-fee AI execution on Hyperliquid — CONDITIONAL BET
- **What:** AI-native execution/automation + risk layer; earn Hyperliquid **builder fees**
  (≤10 bps perps) on routed volume; never custody funds; fee → Base token buyback.
- **Money (verified):** ecosystem paid **$40M+**; PVP.trade **$7.2M** lifetime; Phantom
  **>$100k/day**; ~40% of HL users trade via 3rd-party frontends. Self-funds from first
  routed volume.
- **AI role:** NOT alpha/signals (that's hype that blows up). Real edge = natural-language
  → execution, autonomous risk/limit enforcement, strategy explainability.
- **Niche:** non-US semi-pros; vault & HIP-3 automation; regional/non-English. Generic
  bot = loses.
- **Build:** solo, 8–12 wk, <$3k. Non-custodial via agent keys.
- **Biggest risks:** Hyperliquid platform concentration (share swung 80%→20% in 2025 →
  design multi-venue); crowded; US geo-blocked; "AI = alpha" framing would kill cred.

## H1 — "Quant Desk": AI-managed non-custodial Morpho vault — CONDITIONAL BET
- **What:** Morpho **curator** on Base in **volatile/emerging collateral** incumbents
  won't touch; AI = real-time LLM synthesis of unstructured risk (governance/oracle/
  exploit/social) → faster cap adjustments + **radical public risk logs** as the trust
  moat. Non-custodial (curator allocates, can never withdraw).
- **Money:** 1% mgmt + 10–20% perf → ~$16k/yr @ $1M TVL; ~$160k @ $10M; ~$1.6M @ $100M.
  **Breakeven ≈ $10M TVL; ~$0 for the first ~6 months.**
- **Audit/cost:** inherits Morpho's audited contracts (no $80k+ vault audit); only a token
  contract needs ~$5–20k. Infra <$500/mo.
- **Niche:** Base-native risk-on collateral, EUR stables, curator-less small L2s — gaps
  the $2B incumbents (Steakhouse, Gauntlet, Sentora) ignore.
- **Build:** solo, 10–14 wk. Keeper circuit-breakers non-negotiable.
- **Biggest risks:** "AI-washing" crowd (Almanak, Brahma Morpho Agent, Shogun, ARMA) →
  need a demonstrable risk-catch event within 6mo; 18–30 mo grind to real revenue;
  exploit/rogue-keeper = business-ending; depositor-trust moat favors incumbents.
